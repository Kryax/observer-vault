# Two Antagonistic Modes

## Structural Description

A system oscillates between two large-scale operating states that are mutually suppressive — when one is active, the other is dampened. Each mode serves a fundamentally different function, and the system cannot fully occupy both simultaneously. Health requires regular alternation between the modes. Pathology manifests as getting stuck in one mode or losing the ability to transition cleanly. The transition between modes is itself a regulated event, not a gradual blend.

## Domain-Independent Formulation

A system with two mutually suppressive operating modes that serve opposed functions, where health requires controlled alternation and pathology is mode-lock.

## Instances

### Instance 1: Cognitive Science — Default Mode vs Task-Positive Network
- **Domain:** Cognitive Science / Neuroscience
- **Expression:** The brain's default mode network (rest, self-referential processing, mind-wandering) and task-positive network (focused external engagement, goal-directed action) anti-correlate. When one activates, the other is suppressed. The transition between them is regulated — inability to disengage the default mode is associated with depression; inability to engage it is associated with burnout. Neither mode is superior; the system needs both.

## Discovery Context

Discovered during a slow triad exploring cognitive science. The default mode network / task-positive network anti-correlation was the clearest instance, but the pattern of mutually suppressive operating modes requiring controlled alternation was visible across multiple cognitive phenomena (encoding vs consolidation in memory, Hebbian strengthening vs homeostatic scaling in plasticity).

## Falsification Conditions

- If supposed "antagonistic" modes can be shown to operate simultaneously without interference in well-functioning systems (i.e., mutual suppression is incidental rather than structural), the motif would be superficial.
- If the two-mode structure consistently reduces to a continuous spectrum with no discrete mode-switching (i.e., systems smoothly blend rather than toggle), the motif would be describing a false dichotomy.
