# R-Axis Tier 2 Status

## Current Earned Interpretation

The current earned interpretation is that the recurse-axis Tier 2 search has not produced a clean standalone structural operator. The search has produced real signal, but that signal has repeatedly collapsed into overlap with existing differentiate-axis and integrate-axis Tier 2 motifs. The strongest current read is that recurse-axis behavior may be emergent and compositional rather than represented by a missing peer operator at Tier 2.

## Core Conclusions

### Substrate Bias Confirmed

- GitHub evidence has been effect-heavy: it shows what systems do to reachable futures, rule surfaces, or affordance fields, but often only indirectly exposes the mechanism.
- arXiv evidence has been mechanism-heavy: it more often names projections, interfaces, filters, coupling laws, boundary conditions, and mathematical control terms that produce the downstream change.
- Neither substrate natively surfaces reflective or cybernetic recurse vocabulary.
- Three zero-result reflective `cs.AI` scrapes are now empirical evidence of this bias:
  - `self-referential OR autopoiesis OR metacognition OR self-modifying` -> `0`
  - `second-order cybernetics OR observer-dependent OR participatory` -> `0`
  - `process philosophy OR enactivism OR embodied cognition` -> `0`
- This means the evidence substrate is shaping the apparent motif search space. Mechanism is easier to retrieve than reflective/meta-structural framing.

### RSR тАФ Reachability-Space Rewriting

- `Reachability-Space Rewriting` is now reclassified as an effect descriptor, not a standalone operator.
- This reclassification was first forced by adversarial GitHub-domain testing and then confirmed by arXiv.
- Across both evidence classes, `RSR` remains useful as an annotation for cases where futures open/close, but it does not survive `d` as a non-derivable Tier 2 motif.

### AFR тАФ Admissibility-Field Reparameterization

- `Admissibility-Field Reparameterization` is a stronger cause-shaped candidate than `RSR`.
- It captures the recurring move where projections, interfaces, policy terms, filters, coupling laws, or boundary conditions change which transitions count as available or stable.
- But `d` remains contested.
- The strongest overlaps are with:
  - `Dual-Speed Governance`
  - `Alignment by Compression`
  - gating / chokepoint / explicit state-structure families
- So AFR is a better explanatory object than RSR, but still not cleanly non-derivable enough to stand as earned Tier 2.

### Tier 2 Standalone Recurse Operator Now Doubtful

- Multiple candidate formulations have been tested.
- Each formulation has either:
  - collapsed into effect-description (`RSR`), or
  - collapsed into overlap with existing D-axis and I-axis Tier 2 operators (`AFR` and adjacent cause-level framings).
- This does not mean recurse-axis behavior is absent.
- It does mean the search for a standalone Tier 2 recurse peer is now doubtful.

### Tier 3 Compositional Hypothesis Is Current Best Read

- The best current interpretation is that recurse-axis behavior may emerge from how existing Tier 2 motifs compose.
- In this reading, recurse is not missing because it has failed to be found; it may be missing because it is not a standalone operator class at that tier.
- Instead, recurse may appear when existing structural operators interact such that admissibility, framing, or future-space gets rewritten downstream.
- This is now the next frontier.
