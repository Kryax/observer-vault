# Recursive Generativity

## Structural Description

A system whose recursive operation does not merely repeat or refine but generates genuinely novel structure at each iteration. The output of each recursive pass is not a better version of the input but a qualitatively different kind of thing. The recursion crosses levels — it doesn't loop within a level but ascends through them, producing the conditions for further ascent.

## Domain-Independent Formulation

Recursion that generates novel structure at each pass rather than refining existing structure — each iteration produces the substrate for a qualitatively different next iteration.

## Instances

### Instance 1: Mathematical Logic
- **Domain:** Foundations of Mathematics
- **Expression:** Gödel's incompleteness — each formal system, when it recurses on itself (proves statements about its own provability), generates statements that require a new system to resolve. The recursion generates the next level.

## Falsification Conditions

- A recursive system that always produces the same kind of output regardless of iteration depth (pure refinement, never level-crossing)
- Evidence that the apparent level-crossing is illusory and all iterations operate within the same structural level
- A domain where the pattern appears but the "novel structure" is predictable from the prior level (not genuinely emergent)
