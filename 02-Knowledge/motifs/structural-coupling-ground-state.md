# Structural Coupling as Ground State

## Structural Description

The most basic form of integration is not active synthesis but passive structural coupling — two systems that share an interface become mutually constrained simply by coexisting. No agent performs the integration; the coupling is the ground state that exists before any intentional integration occurs. All higher-order integration (active synthesis, deliberate composition, hierarchical binding) builds on top of this passive coupling substrate.

## Domain-Independent Formulation

Integration at its zeroth order is passive structural coupling: mutual constraint arising from shared interface, prior to any intentional binding operation.
