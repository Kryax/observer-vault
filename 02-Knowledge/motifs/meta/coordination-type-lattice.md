# Coordination Type Lattice

## Structural Description

The space of Tier 2 structural operators is organized as a partial order (lattice) where each operator addresses a type of coordination problem whose necessity is generated by solving the previous type. The operators form a directed acyclic graph: boundary coordination (what happens at resource limits) → sequence coordination (what follows what) and composition coordination (how independent parts combine, unordered with respect to sequence) → convergence coordination (how distributed parts reach agreement) → temporal coordination (how fast and slow decisions interact) → reflexive coordination (how the coordination framework itself evolves). Each level subsumes its predecessors: a system capable of reflexive coordination must also be capable of all five lower coordination types.

The lattice has a diamond structure at the differentiate axis (sequence and composition coordination are independent prerequisites for convergence coordination) and a linear chain elsewhere. The apex of the lattice (reflexive coordination / OFL) has different metric structure from the lower nodes: derivative order becomes interval-valued rather than point-valued at the apex, because self-reference collapses the distinction between the process being described and the meta-process that generates it.

## Domain-Independent Formulation

Structural operators inhabit a partial order defined by coordination type, where solving each coordination problem reveals the next, and the lattice's apex exhibits topological irregularity because reflexive coordination collapses the depth metric that stratifies the lower levels.

## Discovery Context

Emerged from a slow triad (Oscillate → Converge → Reflect) targeting Tier 3 meta-structural analysis across the six Tier 2 operators (DSG, CPA, ESMB, BBWOP, ISC, OFL). Three perspective agents (compositional, geometric, generative) independently converged on ordered generative dependencies between operators. The generative agent proposed the coordination type chain; the compositional agent found the three-layer decomposition that maps onto the lattice; the geometric agent found the cascade/alternative grammar that the lattice explains.

Prior work (March 11 Tier 3 composition inquiry) established that recurse-axis behavior is compositional, requiring DSG × ESMB × CPA at minimum. The lattice subsumes this finding: OFL sits above DSG, ESMB, and CPA in the partial order, explaining WHY those three are the minimum prerequisites for recursion.

## Tier 3 Criteria Assessment

1. **Geometric description:** MET. The lattice is a geometric structure (partial order) describing how operators relate through generative prerequisites.

2. **Tier-independent falsification:** PARTIALLY MET. If one operator is demoted, the lattice loses a node but the remaining partial order holds. Coordination type labels are derived from the operators (post-hoc naming concern). The falsification conditions below address this.

3. **Self-referential prediction:** MET. Predicts:
   - A minimal element with no prerequisites (BBWOP confirmed)
   - A maximal element requiring all others (OFL confirmed)
   - Incomparable pairs (ESMB ∥ CPA confirmed)
   - That the motif library's own development should traverse the lattice (early sessions discovered D-axis operators; later sessions discovered I-axis; R-axis came last)
   - Gap predictions: the "join" of any two parallel operators should exist (ISC as join of ESMB and CPA is testable)

4. **Load-bearing, not decorative:** PARTIALLY MET. Makes specific predictions:
   - OFL should not appear in systems lacking DSG-like temporal separation
   - BBWOP should appear in even very simple systems
   - Adding more D-axis operators at the base of the lattice adds breadth but not new coordination types
   - The post-hoc labeling concern weakens the load-bearing claim: if the coordination types cannot be derived independently of the operators they describe, the lattice may be a sophisticated description rather than a generative theory

## Open Issues (Gaps Preventing Promotion)

1. **Post-hoc naming:** The coordination type labels (boundary, sequence, composition, convergence, temporal, reflexive) are derived FROM the operators. If these labels could be derived from first principles (e.g., from the D/I/R axis structure plus derivative order), the lattice would be significantly stronger. Currently, there is a circularity risk: the lattice "predicts" the operators because it was built from them.

2. **Cross-domain lattice testing:** The lattice has been checked against biology (immune system — full traversal) and simple systems (BBWOP only). It needs testing against more domains to confirm that the prerequisite ordering holds universally, not just in software-derived systems.

3. **Independence from axis closure:** The lattice might reduce to the simpler Axis Closure Principle (self-amendment requires D + I + R) plus the observation that D operators come first developmentally. If the lattice's predictions reduce to axis closure's predictions, it is not a separate Tier 3 motif.

4. **The ESMB ∥ CPA independence claim:** Agent 3 proposed ESMB and CPA are incomparable (neither requires the other). This needs adversarial testing: are there systems with ESMB but no CPA that function well? And vice versa? If one always requires the other, the lattice's diamond collapses to a chain.

## Falsification Conditions

- If OFL is found operating in systems that have never had boundary coordination (BBWOP-like), temporal coordination (DSG-like), or differentiation (ESMB/CPA-like), the lattice's prerequisite ordering is wrong
- If ESMB and CPA always co-occur (never found independently in systems), the diamond structure is wrong — they are not independent prerequisites
- If the coordination type labels cannot be derived from any framework independent of the operators they label, the lattice is a post-hoc description, not a generative theory — it fails the load-bearing criterion
- If systems routinely skip levels in the lattice (e.g., acquire DSG before ISC without any convergence mechanism), the ordering is wrong
- If a new Tier 2 operator is discovered that addresses a coordination type NOT predicted by the lattice's gap structure, the lattice is incomplete as a geometry of the operator space
